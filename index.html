HTML (HyperText Markup Language)

  • HTML itu seperti kerangka dasar sebuah rumah. Dia yang membentuk struktur dasar dari website
  • Tanpa HTML, konten website tidak akan terorganisir dengan baik
  • HTML memungkinkan kita menampilkan teks, gambar, video, link, dan elemen-elemen dasar lainnya
  • Sangat penting karena mesin pencari seperti Google membaca HTML untuk memahami isi website

CSS (Cascading Style Sheets)

  • CSS itu seperti cat dan dekorasi rumah. Dia yang membuat website jadi cantik dan menarik
  • Tanpa CSS, website akan terlihat sangat polos dan membosankan
  • CSS mengatur tampilan seperti warna, ukuran, jarak, tata letak, dan animasi sederhana
  • Membuat website jadi responsive (bisa menyesuaikan dengan berbagai ukuran layar)

JavaScript

  • JavaScript seperti sistem elektronik dalam rumah yang membuat segala sesuatu bisa bergerak dan berinteraksi
  • Membuat website jadi lebih hidup dan interaktif
  • Bisa membuat fitur-fitur canggih seperti validasi form, animasi kompleks, dan update konten tanpa refresh
  • Memungkinkan website berkomunikasi dengan server untuk mengambil atau mengirim data

Cara Dasar Integrasi

  • HTML sebagai struktur dasar
  • CSS ditulis di dalam tag <style> atau file terpisah (.css)
  • JavaScript ditulis di dalam tag <script> atau file terpisah (.js)
  • Ketiganya saling terhubung dan bekerja sama untuk membentuk website yang lengkap

Cara Kerja Bersama

  • HTML menyediakan struktur dan konten
  • CSS akan menghias elemen-elemen HTML
  • JavaScript akan menambahkan interaktivitas pada elemen-elemen tersebut

Best Practice

  • Pisahkan ketiga bahasa ini dalam file terpisah untuk kode yang lebih rapi
  • HTML fokus pada struktur konten
  • CSS fokus pada tampilan
  • JavaScript fokus pada fungsi dan interaktivitas
  • Gunakan komentar untuk dokumentasi kode
  • Terapkan prinsip "separation of concerns" (pemisahan tanggung jawab)
